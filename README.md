# GithubSearchApp

GitHub 유저 검색 API를 활용한 iOS 어플리케이션.

## Description
- 최소 타겟 : iOS 13.0
- Portrait 모드만 지원
- MVVM - Input/Output 패턴 적용
- Storyboard를 활용하지 않고 코드로만 UI 구성
- Swiftlint 적용으로 코드 컨벤션 확립
- 라이트 모드/다크 모드 대응
- Pagination 구현
- [개발 공수](https://maze-mozzarella-6e5.notion.site/6ba9d82a6950406092d5785e30ac2da3)

## Feature
* 유저 검색 뷰
  + 유저 실시간 검색 기능
  + 즐겨찾기 추가/제거
  + 페이지네이션
  + 액티비티 인디케이터
* 즐겨찾기 유저 뷰
  + 즐겨찾기 유저 실시간 검색 기능
  + 검색어 색깔 변환
  + 즐겨찾기 유저 제거
